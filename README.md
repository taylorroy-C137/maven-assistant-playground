# Maven Consumer Playground

A playground for viewing and building Maven prototypes — Consumer screens, Maven Assistant conversations, and more.

**Live site:** [maven-assistant-playground.vercel.app](https://maven-assistant-playground.vercel.app)

**Setup guide:** Once logged in, click "Get Started" in the header — or go to [/getting-started](https://maven-assistant-playground.vercel.app/getting-started) for the full walkthrough.

## Quick start (if you already have Cursor and Node.js)

```
git clone https://github.com/taylorroy-C137/maven-assistant-playground.git
cd maven-assistant-playground
npm install
npm run dev
```

Then open [localhost:3000](http://localhost:3000) in Chrome.
